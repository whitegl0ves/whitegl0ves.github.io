Vue 在解析模板字符串时，可分为两种情况：以 < 开头的字符串和不以 < 开头的字符串。

其中，不以 < 开头的字符串有两种情况：它是文本节点或 {{ exp }} 插值表达式。

而以 < 开头的字符串又分为以下几种情况：

  1. 元素开始标签，比如 <div>
  2. 注释节点 <!-- 123 -->
  3. 文档声明 <!DOCTYPE html>
  4. 纯文本标签 <![CDATA[<]]>
